# Hasanov_Natik — бот статусов рейсов AZAL

## Что делает
Присылаете боту номер рейса (например `J2123` или `123`) —
он ищет его на сайте airport.az (вылеты + прилёты) и присылает статус:
вовремя / задержка / отмена / номер терминала / гейт.

## Запуск на Railway — 5 шагов

1. Загрузите оба файла (`bot.py`, `requirements.txt`, `Procfile`) в ваш
   GitHub-репозиторий (или в существующий проект Railway, если он подключён к репо).

2. В Railway откройте свой проект → сервис → вкладка **Variables**.

3. Добавьте переменную:
   - Name: `BOT_TOKEN`
   - Value: ваш токен от @BotFather (тот, что получили ПОСЛЕ Revoke)

4. Railway сам определит, что это Python-проект, установит зависимости
   из `requirements.txt` и запустит `bot.py` (через `Procfile`).

5. Откройте бота в Telegram, напишите `/start`, затем пришлите номер рейса.

## Если бот не отвечает / ошибка при поиске рейса

Сайт airport.az может немного менять разметку страницы — если парсер
перестанет находить рейсы, пришлите мне:
- скриншот ошибки из логов Railway (вкладка **Deployments** → открыть лог)
- и я обновлю код под изменения сайта.

## Важно по безопасности
Никогда не публикуйте `BOT_TOKEN` в открытом репозитории или чате.
Если токен случайно "засветился" — сразу делайте Revoke в @BotFather.
